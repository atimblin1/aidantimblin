# Aidan Timblin – Personal Website

This repository contains the source for my personal website, hosted via GitHub Pages.

The site is a simple, single-page portfolio that highlights:
- My work as a Junior Web Developer
- Featured projects
- Technical skills
- Contact information

The site is intentionally lightweight and uses only static assets.

## Tech
- HTML
- CSS
- No frameworks

## Files
- `index.html` – Main site content
- `styles.css` – Styling
- `sitemap.xml` – Search engine indexing
- `robots.txt` – Crawler instructions
